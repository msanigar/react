# React app

Learning React JS, React-Router, Redux, Karma (& Mocha/Chai)

# Set up
Clone: `https://github.com/myles91/react`

``` bash

cd React

npm i -g webpack karma-cli mocha
npm i
gulp

```

# Tests

Single run, headless:

``` bash

gulp test

```

Watch tests, headless:

``` bash

gulp test:watch

```

Single run, all browsers:

``` bash

gulp test:browsers

```
